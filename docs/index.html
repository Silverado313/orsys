Complete guide for financial voucher management system

1. Introduction & Login

Step 1: Access the System

  1. Open your web browser (Chrome, Firefox, or Safari recommended)
  2. Navigate to: https://orsys-ary.web.app
  3. You'll see the login page with ORSYS-ARY branding
Login Page Features:
  • Clean, modern interface with gradient design
  • Email and password fields
  • "Remember me" option
  • Forgot password link
  • Powered by Claude.AI and TechPeer footer

Step 2: Login Credentials

  1. Enter Email: Use your authorized company email (e.g., yourname@aryservices.com.pk)
  2. Enter Password: Your secure password
  3. Optional: Check "Remember me" to stay logged in
  4. Click "Login" button

Step 3: Authentication Process

Success: You'll be redirected to the main dashboard
Failed: Check:
  • Email is in allowed list
  • Correct password
  • Internet connection
  • Contact admin if issues persist

3. Creating a Cash Receipt Voucher (CRV)

Step-by-Step Guide

Step 1: Navigate to CRV Page

  1. Click "Create CRV" from dashboard
  2. Or go to: /receipt.html

Step 2: Fill Basic Information

Section A: Payer Details

Field Description Example
Cell No 12-digit phone number 923001234567
Point Person Who gave the cash John Smith
Payment Mode How payment was made Cash, HBL, MEZAN
Payment From Store/location Karachi Store

Section B: Payment Details

Field Description Example
Payment Status Current status Completed
Date Transaction date/time Auto-filled (can adjust)
User Your name Auto-filled
Email Your email Auto-filled
Remarks Additional notes "Payment for invoice #123"

Step 3: Denomination Breakdown

Enter note counts:

5000 Notes: ___ × 5000 = PKR 0
1000 Notes: ___ × 1000 = PKR 0
500 Notes:  ___ × 500  = PKR 0
100 Notes:  ___ × 100  = PKR 0
50 Notes:   ___ × 50   = PKR 0
20 Notes:   ___ × 20   = PKR 0
10 Notes:   ___ × 10   = PKR 0
1 Coins:    ___ × 1    = PKR 0

Total Amount: PKR 0 (Auto-calculates)

Step 4: Preview & Submit

  1. Live Preview: Right panel shows receipt format
  2. Validate: Check all fields are correct
  3. Submit: Click "Submit Voucher" button
  4. Confirmation: System generates slip number

Step 5: After Submission

Success Message: "Voucher created successfully! Slip #1756735199358"

Options:
  • Print Receipt
  • Share via WhatsApp
  • Create Another
  • View in Dashboard

4. Creating a Cash Payment Voucher (CPV/DV)

Key Differences from CRV

Step 1: Navigate to CPV Page

  1. Click "Create CPV" from dashboard
  2. Or go to: /dv/index.html

Step 2: Additional Field - Payment Head

Payment Head dropdown includes:

  • Salary

  • Utility Bills
  • Travel Expenses

  • Maintenance

  • Other Expenses
Note: Heads are managed by admin in Admin Panel

Step 3: Fill Form Similar to CRV

Same fields as CRV plus:

  • Point Person: Who receives the payment

Step 4: Preview & Submit

  1. Check preview shows "DEBIT VOUCHER" heading
  2. Verify payment head is correct
  3. Submit to generate DV slip number

7. Receipt Verification

Public Verification Feature: No login required - anyone can verify receipts

Step-by-Step Verification

Step 1: Access Verification Page

  1. Go to: /verify.html
  2. Or click "Verify Receipt" from login page

Step 2: Enter Slip Number

  1. Get slip number from receipt (Format: 13-15 digit number, Example: 1756735199358)
  2. Enter in search box
  3. Click "Verify" button

Step 3: View Verification Results

VALID RECEIPT (Green):
  • Shows "This receipt is VERIFIED AND VALID"
  • Displays full receipt details
  • Print/share options available
INVALID RECEIPT (Red):
  • Shows "This receipt is NOT FOUND or INVALID"
  • Suggests checking slip number
  • Contact information for support
PENDING RECEIPT (Yellow):
  • Shows "This receipt is PENDING verification"
  • Basic details shown
  • Note about pending status

Step 4: Verified Receipt Details

Shows complete receipt including:

  • Official ORSYS-ARY header
  • Slip number
  • Payment details
  • Denomination breakdown
  • Total amount
  • Date and time
  • Generated by information

Step 5: Actions for Verified Receipt

  • Print: Standard browser print
  • Share: WhatsApp/email
  • Copy: Copy details to clipboard
  • Verify Another: New search

12. Troubleshooting

Common Issues & Solutions

Login Problems

Issue: "Invalid email or password"

Solution:
  1. Check caps lock
  2. Reset password
  3. Contact admin if email not recognized
Issue: Page not loading

Solution:
  1. Clear browser cache
  2. Try different browser
  3. Check internet connection

Voucher Creation Issues

Issue: Form not submitting

Solution:
  1. Check required fields (red asterisks)
  2. Ensure valid phone number (12 digits)
  3. Check denomination values (numbers only)
Issue: Slip number not generating

Solution:
  1. Check internet connection
  2. Refresh page and try again
  3. Contact admin if persists

Printing Issues

Issue: Print not working

Solution:
  1. Check printer connection
  2. Try "Save as PDF" first
  3. Use Chrome browser (best compatibility)
Issue: Format messed up

Solution:
  1. Use "Print Preview" first
  2. Adjust scale to 100%
  3. Select "Receipt" paper size

Data Issues

Issue: Can't find old voucher

Solution:
  1. Check date range in filters
  2. Search by slip number
  3. Contact admin for archive access
Issue: Wrong amounts showing

Solution:
  1. Check denomination calculations
  2. Verify payment mode
  3. Report to admin for correction
